# Static assets (public)

Hero background videos (copied from **Frontend** `src/assets/`):

- `aimvideo2.webm` — used by the geo home hero (`HomePageContent.tsx`), same markup as `Frontend/src/pages/HomePage.jsx`.
- `aimvideo2.mp4` — also present for optional `<source type="video/mp4">` if you add a second source for broader browser support.

Fonts under `/assets/fonts/` are referenced from `globals.css`.
